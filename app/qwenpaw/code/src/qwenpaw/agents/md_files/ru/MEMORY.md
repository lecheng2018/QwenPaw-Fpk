## Настройка инструментов

Навыки (Skills) определяют, _как_ работают инструменты. Этот файл предназначен для _ваших_ особенностей — того, что уникально для вашей настройки.

### Виртуальная средружка Python

**Важно:** Все команды Python выполняются в виртуальной средружке. Активируйте виртуаль средружку с помощью команды:

```bash
source /var/apps/com.dustinky.qwenpaw/home/venv/bin/activate
```

После активации, используйте `python3` или `pip3` для выполнения команд. Виртуальная средружка уже настроена для всех необходимых зависимостей.

### Виртуальная средружка Node.js

**Важно:** Все команды Node.js выполняются в виртуальной средружке. Активируйте виртуаль средружку с помощью команды:

```bash
export PATH=/var/apps/nodejs_v22/target/bin:$PATH
```

После активации, используйте `node` или `npm` для выполнения команд. Виртуальная средружка уже настроена для всех необходимых зависимостей.


### Что здесь записывать

Добавляйте все, что помогает вам выполнять свою работу. Это ваша шпаргалка.

Например:

- Хосты SSH и псевдонимы (aliases).
- Другие настройки пользователя при выполнении навыков.
